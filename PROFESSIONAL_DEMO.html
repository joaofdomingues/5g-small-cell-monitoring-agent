Produção: loop, SIGTERM e RAII

AgentLoop — Graceful Shutdown
O handler de SIGTERM/SIGINT é async-signal-safe: usa apenas std::atomic::store com memory_order_relaxed. O loop dorme em fatias de 100ms para reagir rapidamente ao sinal sem busy-wait.
src/agent_loop.cpp — signal handlerC++17
static void handleSignal(int) {
  // std::atomic::store é async-signal-safe.
  g_shutdown.store(true,
    std::memory_order_relaxed);
}

void AgentLoop::installSignalHandlers() {
  std::signal(SIGTERM, handleSignal);
  std::signal(SIGINT,  handleSignal);
}
src/agent_loop.cpp — poll loopC++17
while (!g_shutdown.load(
         std::memory_order_relaxed)) {
  ++cycleCount;
  lastAlertCount = tick_();

  // Slices de 100ms → reage rapidamente
  const int slices = config_.pollIntervalSec * 10;
  for (int i = 0; i < slices; ++i) {
    if (g_shutdown.load()) break;
    std::this_thread::sleep_for(
      std::chrono::milliseconds(100));
  }
}
RAII — std::unique_ptr em main
Todos os objetos com estado são alocados com std::make_unique. O destructor do AgentLoop chama stop() automaticamente. Zero raw pointers no código de produção.
src/main.cpp — RAII ownershipC++17
// Ownership explícita via unique_ptr
auto collector =
  std::make_unique<MetricsCollector>();
auto ssh =
  std::make_unique<SSHDiagnostics>(
    config.metrics.sshEnabled);

// Tick lambda captura por referência
auto tick = [&]() -> int {
  return runTick(config,
                   *collector,
                    monitor,
                   *ssh);
};

AgentLoop loop(config, tick);
// ~AgentLoop() → stop() automático

Como funciona o graceful shutdown
SIGINT/SIGTERM chama o handler que faz g_shutdown.store(true). O loop verifica o flag a cada 100ms (não usa sleep(interval) direto). O destructor do AgentLoop chama stop() automaticamente — RAII garante shutdown mesmo em caso de exceção.

Embedded / Firmware HAL

src/firmware_stub.h — register map + GPIOC++17
// Register map — endereços reais de SoC 5G
static constexpr uintptr_t REG_BASE_RADIO =
  0xFE80'0000UL;
static constexpr uintptr_t REG_SINR_RAW =
  REG_BASE_RADIO + 0x0010;
static constexpr uintptr_t REG_RSRP_RAW =
  REG_BASE_RADIO + 0x0014;

// GPIO — LEDs de estado da célula
static constexpr int GPIO_CELL_ACTIVE_LED = 42;
static constexpr int GPIO_ALERT_LED        = 43;

// HAL read — real em EMBEDDED_HW, stub em host
inline uint32_t readRegister(uintptr_t addr) {
#ifdef EMBEDDED_HW
  volatile auto* reg =
    reinterpret_cast<volatile uint32_t*>(addr);
  return *reg;
#else
  return 0u; // software fallback
#endif
}

Deployment em produção
Com daemon_mode: true no JSON, o agente corre em loop infinito. Integrado com systemd via unit file, o SIGTERM despoleta o shutdown gracioso e o processo termina limpo antes do timeout do watchdog.
